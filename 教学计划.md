### 第6节：条件判断 + 循环 — 让电脑学会思考（1.5h）

**前半段（~45min）— 条件判断**：
- `if / elif / else` — 做判断
- 比较运算符：`>`, `<`, `==`, `!=`
- 🎮 做一个"成绩评价器"：输入分数 → 输出评语

**后半段（~45min）— 循环 + 猜数字游戏**：
- `while` 循环 — 一直重复直到猜对
- `import random` — 生成随机数
```python
import random
answer = random.randint(1, 100)

while True:
    guess = int(input("猜一个1-100的数字："))
    if guess > answer:
        print("太大了！")
    elif guess < answer:
        print("太小了！")
    else:
        print("猜对了！你真厉害！")
        break
```

**课后产出**：猜数字完整游戏，push 到 GitHub

### 第7节：列表 + for 循环（1.5h）

**前半段（~45min）— 列表入门**：
- 列表 `[]` — 一个盒子装很多东西
- `append()`, `len()`, 索引访问
- 🎮 做一个"我最喜欢的游戏"排行榜

**后半段（~45min）— for 循环 + 函数**：
- `for item in list` — 遍历每个东西
- `def` — 定义自己的函数，像整理收纳盒
- 🎮 做一个"待办清单"命令行工具

```python
todos = []

def add_todo(task):
    todos.append(task)
    print(f"✅ 已添加：{task}")

def show_todos():
    print("\n📋 我的待办清单：")
    for i, task in enumerate(todos, 1):
        print(f"  {i}. {task}")

add_todo("写作业")
add_todo("学Python")
show_todos()
```

**课后产出**：待办清单工具 + push 到 GitHub

### 第14节：机器学习入门 — 教电脑识别东西（1.5h）

> 🧠 科普+体验课，不要求完全理解

**前半段（~45min）— 感性认知**：
- 什么是机器学习？（不是写规则，而是给例子）
- 演示：画几个点，画一条线穿过它们（**线性回归**直觉）
- 用 `sklearn` 做一个最简单的预测（输入成绩 → 预测是否及格）

```python
from sklearn.linear_model import LinearRegression
import numpy as np

# 学习小时数 vs 考试分数
X = np.array([[1], [2], [3], [4], [5], [6]])
y = np.array([50, 55, 65, 70, 80, 85])

model = LinearRegression()
model.fit(X, y)
print(f"如果学7小时，预计考 {model.predict([[7]])[0]:.0f} 分")
```

**后半段（~45min）— 图像识别初体验**：
- 什么是像素？一张图 = 很多数字
- 什么是卷积？用 NumPy 演示一个小矩阵"滑动窗口"
- 什么是 Max Pooling？每 2×2 格子只保留最大值 → 图片缩小
- 可以用 Agent 帮忙写演示代码

**课后产出**：理解了 ML 的基本直觉
